# \# 📂 File Approval System

# 

# A simple and clean \*\*ASP.NET Core MVC (.NET 8)\*\* application that manages file approvals through a multi-step workflow.

# 

# \---

# 

# \## 🚀 Features

# 

# \- 🔐 Authentication (Session-based login)

# \- 📁 File Management (Create / List / Upload)

# \- ✅ Multi-level Approval Workflow:

# &#x20; - Employee 1 → Creates file

# &#x20; - Employee 2 → First approval

# &#x20; - Employee 3 → Final approval

# \- 🔍 Search \& Filtering (File Number, Subject, Category, Status)

# \- 📊 Dashboard with statistics

# \- 📄 File Attachment Upload \& Download

# \- 📑 Pagination

# \- 🎨 Modern UI using Bootstrap

#
